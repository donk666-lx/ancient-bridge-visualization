# 中国古代桥梁数据报告 · 沉浸式可视化

一个单文件 HTML 沉浸式数据可视化项目，展示中国古代桥梁的历史、地域、技术与文化价值。

![预览](screenshot.png)

---

## 项目概述

本项目基于茅以升《中国古代桥梁史》等权威资料，收录 **25 座代表性古桥**，通过 6 大篇章、多维度数据可视化，呈现中国古代桥梁从隋代至民国的千年发展史。

### 六大篇章

| 篇章 | 主题 | 核心内容 |
|------|------|----------|
| S1 封面 | 印象篇 | 水墨动效、四大统计数字（25座/7朝代/15省/1400+年） |
| S2 地域分布 | 空间篇 | 中国地图热力图、双环饼图、省份排行 |
| S3 历史时间 | 朝代篇 | 7 朝代选择条、柱状+折线组合图、世界对比 |
| S4 结构类型 | 技术篇 | 雷达图对比 5 种桥型、点击卡片看详情 |
| S5 文化价值 | 人文篇 | 30 词云、桑基图、8 则历史故事 |
| S6 彩蛋总结 | 荣誉篇 | 6 张纪录卡、粒子动画、Canvas 海报生成器 |

## 技术栈

| 技术 | 用途 |
|------|------|
| **原生 HTML/CSS/JS** | 单文件架构，零构建 |
| **ECharts 5.4.3** | 地图、饼图、柱状图、雷达图、桑基图 |
| **GSAP 3.12.2** | 入场动画、时间线控制 |
| **Noto Serif SC** | 思源宋体，书法质感 |
| **Canvas API** | S6 粒子效果、海报生成 |

### 设计系统

- **主色调**: 宣纸米白 `#f5f0e8`、浓墨 `#1a1008`、赭石 `#c47c2b`
- **朝代色**: 隋(古金)、唐(金黄)、宋(宝蓝)、元(墨绿)、明(明红)、清(紫罗兰)、民国(橘红)
- **字体**: Noto Serif SC（中文衬线）、system-ui（数字）

## 使用方式

### 直接打开

```bash
# 用浏览器直接打开（推荐 Chrome/Edge/Firefox）
index.html
```

**注意**: 需要联网加载 CDN 资源（ECharts、GSAP、Google Fonts）。

### 本地开发

```bash
# 项目结构
.
├── index.html          # 主文件（76KB，自动生成）
├── page_structure.md   # 项目规划书 / 设计规范
├── data/
│   └── bridges_data.json   # 25 座桥梁原始数据
├── sections/           # 6 个模块源文件（并行开发）
│   ├── s1_cover.html
│   ├── s2_regional.html
│   ├── s3_timeline.html
│   ├── s4_types.html
│   ├── s5_culture.html
│   └── s6_summary.html
└── assemble_sections.py    # 组装脚本

# 重新组装
python assemble_sections.py
```

## 交互功能

### 全局导航
- **侧边导航点**: 点击或键盘 ↑↓ 切换篇章
- **进度条**: 左侧赭石色进度指示
- **滚动吸附**: 每屏完整显示，流畅切换

### 各篇交互

| 位置 | 交互 |
|------|------|
| S1 封面 | 数字计数动画、按钮悬浮、滚动提示弹跳 |
| S2 地域 | 地图点击省份、饼图联动高亮 |
| S3 朝代 | 点击朝代按钮/柱子 → 右侧面板切换详情 |
| S4 类型 | 点击 5 张卡片 → 全屏模态框展示结构原理 |
| S5 文化 | 词云关键词点击 → 自动滚动到对应故事 |
| S6 总结 | 「生成分享海报」→ Canvas 绘制 800×450 PNG |

## 开发说明

### 多人并行开发流程

本项目采用「规划书 + 并行开发 + 组装」模式：

1. **规划阶段**: 编写 `page_structure.md` 设计规范
2. **并行开发**: 6 个开发者同时编写 `s1~s6.html`
3. **组装阶段**: 运行 `assemble_sections.py` 合并生成 `index.html`
4. **测试修复**: 统一检查、修复 Bug

### 修复历史

| Bug | 修复 |
|-----|------|
| CSS `content:''` 引号未闭合 | 修复 2 处 |
| `countUp()` 传入 DOM 而非 ID | 改为字符串传参 |
| ECharts formatter 字符串含换行符 | 改用 `\n` 转义 |
| `\!` 转义错误 | 3 处改为 `!` |
| 模板字符串 `${t}` 被破坏 | 修复变量插值 |
