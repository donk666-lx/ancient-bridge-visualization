🏛️ 古桥小助手

快速开始

只需两行代码，即可在任何网页中嵌入古桥小助手：

<!-- 1. 引入脚本 --> <script src="https://your-domain.com/widget/BridgeAssistant.js"></script> <!-- 2. 初始化 --> <script> BridgeAssistant.init({ apiUrl: 'http://localhost:3002', position: 'bottom-right' }); </script>

配置选项

参数 类型 默认值 说明
apiUrl string http://localhost:3002 后端API地址
position string bottom-right 位置：bottom-right, bottom-left, top-right, top-left
voice string xiaoxiao 语音类型
voiceEnabled boolean true 是否启用语音
theme string default 主题：default, dark
width number 320 窗口宽度(px)
height number 400 窗口高度(px)
autoOpen boolean false 是否自动打开
welcomeMessage string 默认欢迎语 欢迎消息内容

API 方法

// 打开聊天窗口 BridgeAssistant.open(); // 关闭聊天窗口 BridgeAssistant.close(); // 切换聊天窗口 BridgeAssistant.toggle(); // 销毁小助手 BridgeAssistant.destroy(); // 程序化发送消息 BridgeAssistant.send('赵州桥的历史');

完整示例

<!DOCTYPE html> <html> <head> <title>我的网站</title> </head> <body> <!-- 页面内容 --> <h1>欢迎来到我的网站</h1> <!-- 引入古桥小助手 --> <script src="BridgeAssistant.js"></script> <script> BridgeAssistant.init({ apiUrl: 'http://localhost:3002', position: 'bottom-right', theme: 'default', welcomeMessage: '你好！有什么关于古桥的问题吗？', autoOpen: true }); </script> </body> </html>

特性说明

✅ 完全隔离 - 使用 Shadow DOM 技术，CSS 和 DOM 完全隔离，不会与宿主页面冲突
✅ 响应式设计 - 自动适配移动端和桌面端
✅ 语音合成 - 集成 Edge TTS，支持语音朗读回答
✅ 轻量级 - 纯原生 JavaScript，无需任何依赖
✅ 易于定制 - 支持主题、位置、尺寸等多种配置
